# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/) conventions.

## [1.0.0] — Design Freeze

### Added
- Initial Git version control setup for the project.
- `.gitignore` covering dependencies, build output, environment files, OS/editor artifacts, coverage, and cache — while explicitly keeping documentation, assets, uploads, images, videos, and fonts under version control.
- `/docs` directory containing the full technical documentation set:
  - `PROJECT_ANALYSIS.md` — technical audit of the exported Claude Design handoff bundle.
  - `PROJECT_STRUCTURE.md` — directory tree, page/component map, and routing reference.
  - `DEVELOPMENT_ROADMAP.md` — phased implementation plan for the real build.
  - `CHANGELOG.md` — this file.

### Notes
This commit represents the **frozen visual version** of the Decor Tools website redesign, captured before any CMS or backend development begins. No design, layout, or content changes were made as part of this version-control setup — this release is a snapshot of the design bundle exactly as handed off.
